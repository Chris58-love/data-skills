# Report Expression Rules

Use evidence-bound wording:

- Say "current sample shows" instead of broad market claims.
- Say "can be used as a follow-up validation direction" instead of "must optimize".
- Say "after applying this filter, the sample changed" instead of causal claims.
- Keep low-sample products, segments, or platforms as reference signals, not rankings.

Avoid:

- Absolute rankings without a defined ranking metric and sufficient base.
- Market share, market size, or causality claims without external or experimental evidence.
- Using weak-signal, ad-suspect, or unreviewed records as core proof.
- Introducing outside data in a source-limited report.

Required caveats when applicable:

- Data source coverage.
- Sample size and platform imbalance.
- Manual review status.
- Rule-based tagging limitations.
- Rating or unit conversion assumptions.
- Exclusion/filter impact for branch reports.
